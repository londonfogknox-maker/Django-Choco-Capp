## Overview
This project was built for a Knoxville local author that needed an offical website and include her base of knowledge of different books she recommends utilizing new LLM technology. 

## ‚ú® Features
*   **RAG LLM Moodboard Chatbox**: LLM powered chat bot to explore author's book recommendations made in Python. 
*   **Responsive Design:** Functional on mobile, tablet, and desktop.
*   **Interactive UI:** Smooth transitions and dynamic content using Vanilla JS.
*   **3D Background Effects:** Backgrounds that provide depth and captivating visuals using Three JS.
